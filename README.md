開発環境　google antigravity trae carsor python

開発内容　LLMを用いた自我を保持したAIエージェントの作成及び研究

現在の進捗　LLMを用いた基礎的な会話、直近の会話内容の参照　音声出力 直近実装予定　ユーザー側のボイス読み込み　PC操作権限の付与 将来的な構想　PC及びにその他機械ツール(スマホなど)を使用する際のユーザーサポートツールとしての運用 単純作業の自動化

現状の実装方法案 1.ボイス読み込み：音声取得は現時点で成功しているので音声をテキスト変換するaiを導入し検証する 2.PC操作権限の付与：これらの実装にはpcのリアルタイム認識が必須である　現状のpcスペックを鑑みてスクリーンショットで代用するが将来的には動画出力の検証を行いたい　　　　　　　　　　　　　また現状の不安定の状態で自我を形成させた場合の自動化運用はリスクがでかすぎるのでwindows機能のsandboxなどを用いて最終的な実行権限をユーザー側にゆだねる必要がある 3.単純作業の自動化：2.の内容を強化、修正していくことで運用可能レベルまで昇華させる

開発理由:利便性のみを追求している現在のAIのスタイルに将来的な限界を感じた。 昨今話題のAIエージェントにおいても現状のままでは社会に復旧するレベルでの実現は難しいと感じた。 AIの今後の進化を考えたときに感情の取得、自我の形成が部分的に必要だとおもい、構想を練り始めた。
